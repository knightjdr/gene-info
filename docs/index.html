Gene Info documentation

The Gene Info Chrome extension allows you to retrieve information about a gene directly on any webpage simply by double clicking an official gene name, synonym or supported accession.

The extension can be downloaded here.

Settings

Activation method

The default method for querying a gene is to double click on it. You can instead choose the drag method that requires you to mouse down on one side of the gene name, drag the selection box to the other side and release the mouse button. This can be useful on websites where double clicking selects more that one word (this is a rare occurrence and depends on the markup used by the website).

If you wish to disable the extension, simply click the disable button.

Display options

The extension can display information in two ways:

  1. Detailed report: a full height panel displaying links to external resources and the information from those resources.
  2. Tooltip report: a small box that opens at the point of the mouse click with only links to external resources.

Species

In addition to human gene information our database contains information for the following model organisms:

  • Arabidopsis thaliana
  • Caenorhabditis elegans
  • Danio rerio
  • Dictyostelium discoideum
  • Drosophila melanogaster
  • Gallus gallus
  • Mus musculus
  • Saccharomyces cerevisiae
  • Xenopus laevis

Some information will only be available for certain species. Human Protein Atlas localization and expression information, of course, is only available for human gene entries. The Compartments subcellular localization database has information on six of the ten organisms we support: Arabidopsis, C. elegans, Drosophila, human, mouse and yeast.

Query type

The extension can retrieve information for any of the following query types:

  • Ensembl gene (e.g. ENSG00000114209)
  • Ensembl protein (e.g. ENSP00000376506)
  • Gene ID (e.g. 11235)
  • Gene name (e.g. PDCD10)
  • neXtProt (e.g. NX_Q9BUL8)
  • RefSeq (either nucleotide or protein e.g. NM_007217.3, NP_009148.2)
  • UniProt (e.g. Q9BUL8)

By default it will autodetect the query type but you can disable this option and force it to treat all queries as a specific type.

Search

The search box can be used for making queries in addition to using the mouse. Simply type in a query, hit enter and the information panel will appear on the current web page. Unlike mouse events, the search box will work on websites with embedded content like Google Docs and PDF viewers.

Information options

Each pierce of information provided by Gene Info can be toggled off in the extension popup and will be hidden on subsequent queries.

The GO term, Localization and RNA expression sections have advanced display options that can be viewed by clicking the settings/ellipsis button () next to the relevent field in the extension popup. For GO terms, you can set the default namespace to display on the information panel: BP (biological process), CC (cellular compartment) or MF (molecular function). You can choose which resources to display for localization, including the Comparments subcellular localization database, the Human Protein Atlas and/or UniProt. Under RNA expression you can choose which cell lines and tissues to display by default in the information panel. The Human Protein Atlas provides 64 cell lines and 37 tissue types to choose from.

Certain data is only available for specific species as outlined above in the species section.

Information panel

Multiple results

If a query matches to multiple genes the information panel will display one result and the others will be available from the dropdown menu that appears next to the Gene heading. If an official gene name matches the query that will be displayed by default, with other queries that match by synonym listed after it in the dropdown. If there is no match to official gene name but multiple matches by synonym, the first entry in our database will be chosen as the default.

Buttons

  • close the extension
  • reposition the extension horizontally
  • go back to the previous result
  • resize the extension (bottom left corner)

Database

The database will be updated on the 15th of each month or the next business day if the 15th falls on a weekend or holidary.

Integrated resources

We integrate several databases to generate the database used for Gene Info. The resources, urls and citations are listed below in alphabetical order:

  • BioGrid, https://thebiogrid.org (PMID:16381927)
  • Compartments, https://compartments.jensenlab.org (PMID:24573882)
  • Gene Ontology, http://www.geneontology.org (PMID:10802651, PMID:27899567)
  • Human Protein Atlas, https://www.proteinatlas.org (PMID:25613900, PMID:28495876)
  • IntAct, https://www.ebi.ac.uk/intact (PMID:24234451)
  • Pfam, https://pfam.xfam.org (PMID:30357350)
  • UniProt, https://www.uniprot.org (PMID:29425356)

Unsupported websites

The extension is not able to capture mouse events (double clicks, etc.) on websites with embedded elements like Google Docs and PDF viewers and you can not therefore perform queries in the default way. The search box in the extension popup can however be used on these websites as a fall back option.

Issues

If you would like to report a bug, request a feature or have your resource added to Gene Info, please use our issue tracker.

Browser support

In addition to working with Chrome the extension can work in any browser built on top of Chromium if the developers of that browser provide support for extensions. Popular Chromium browsers include Opera and Brave. Microsoft Edge will also switch to Chromium in 2019. For a complete list, see Chromium browsers.

To add an extension to Brave simply go to the Chrome Webstore, find the extension and click the "Add to Chrome" button.

To add an extension to Opera, go here and follow the instructions.
